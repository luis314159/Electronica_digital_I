Ejercicios de Estudio

1. Compuertas Lógicas

Ejercicio 1.1: Dibuja un circuito que utilice compuertas AND, OR y NOT, y calcula la tabla de verdad para 3 entradas (A, B, C).

Ejercicio 1.2: Diseña un circuito con compuertas XOR y NAND para obtener la expresión lógica (A ⊕ B) ⋅ (A' + B') y realiza su tabla de verdad.

2. Simplificación de Ecuaciones

Ejercicio 2.1 (SOP): Simplifica la siguiente función lógica usando la forma SOP:
F(A, B, C) = A'B'C + AB'C + ABC + A'BC

Ejercicio 2.2 (POS): Convierte la siguiente expresión a POS y simplifícala:
F(X, Y, Z) = (X + Y + Z')(X' + Y + Z)(X + Y' + Z)

Ejercicio 2.3 (Álgebra de Boole): Simplifica la siguiente ecuación aplicando las Leyes de Álgebra de Boole:
F(A, B, C) = A(B + C)(A + B')

3. Mapas de Karnaugh

Ejercicio 3.1: Simplifica usando un mapa de Karnaugh de 3 variables la función:
F(A, B, C) = Σm(1, 3, 5, 7)

Ejercicio 3.2: Utiliza un mapa de Karnaugh de 4 variables para simplificar:
F(W, X, Y, Z) = Σm(1, 3, 7, 11, 15) + d(0, 2, 5)

4. Condiciones "No Importa"

Ejercicio 4.1: Simplifica usando condiciones "No Importa":
F(A, B, C, D) = Σm(1, 3, 5, 7, 9) + d(0, 2, 8)

Ejercicio 4.2: Encuentra la función simplificada usando un mapa de Karnaugh con d(6, 10, 12).

5. Encoder y Decoder

Ejercicio 5.1: Diseña un encoder de 4 a 2 y escribe su tabla de verdad.

Ejercicio 5.2: Diseña un decoder de 2 a 4 y explica cómo funcionan las señales de habilitación.

6. Multiplexores (MUX) y Demultiplexores (DEMUX)

Ejercicio 6.1: Diseña un multiplexor de 4 a 1 y muestra cómo las señales de selección controlan la salida.

Ejercicio 6.2: Diseña un demultiplexor de 1 a 4 e indica cómo se distribuye la señal de entrada entre las salidas.
